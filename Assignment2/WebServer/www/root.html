You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- multiplies two integers (/multiply?num1=3&num2=4)
  • /github -- returns github JSON repo (/github?query=users/amehlhase316/repos)
  • /addition -- adds two integers (/addition?num1=2&num2=9)
  • /subtract -- adds two integers (/subtract?num1=10&num2=2)
